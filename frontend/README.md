# Frontend da Vila Refúgio Hotel

Aplicação React/Vite responsável pela interface do site da Vila Refúgio Hotel.

## Rodar localmente

```bash
npm install
npm run dev
```

URL local padrão:

```text
http://localhost:5173/capsule/
```

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run preview
```

## Stack

- React
- Vite
- Tailwind CSS
- GSAP
- Lenis
- Three.js
- React Icons

## Build

```bash
npm run build
```

O build gera a pasta `dist/`, que deve permanecer fora do Git.
